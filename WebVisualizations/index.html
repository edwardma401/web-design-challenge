Summary: Latitude vs. X


The purpose of the analysis was to identify the relationship between latitude and weather. From the data that contained city-id lists for OpenWeather API in Json format, about 5,000 cities were randomly selected. Then OpenWeather API was used to identify the latitude of the cities and gather corresponding weather information, such as temperature, cloudiness, humidity, and windspeed.


The next step was the visualization of the gathered information. Using matplotlib library of Python, the information were visualized using the scatter plot. Latitude was plotted as y-axis, and other factors such as temperature, humidity, cloudiness, and windspeed were plotted as x-axis. Y-axis near 0 means the equator.
